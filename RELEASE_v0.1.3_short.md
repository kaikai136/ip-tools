# IP诊断工具 v0.1.3

本次版本主要补充了主界面 IP 交互能力，并同步更新了安装包。

## 更新内容

- 支持单击 IP 数字后在主界面直接显示当前选中的 IP。
- 支持双击 IP 数字后打开 Windows CMD，并对该 IP 执行 `ping`。
- 调整主界面选中 IP 展示样式，使其与扫描操作区视觉更统一。
- 重新生成 Windows 安装包与可执行文件。

## 安装包

- `IP诊断工具_0.1.3_x64_zh-CN.msi`
- `IP诊断工具_0.1.3_x64-setup.exe`

## 备注

- 如果安装后桌面图标或快捷方式未立即刷新，通常是系统缓存未更新；重新创建快捷方式或重启资源管理器即可。
