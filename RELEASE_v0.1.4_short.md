# IP诊断工具 v0.1.4

本次版本重点新增了双因子认证工具，并补齐了扫码导入、二维码分享、备份导出和重复项拦截等整套使用流程。

## 更新内容

- 新增“双因子认证”工具入口，支持手动维护 TOTP 条目并实时生成验证码。
- 支持 `otpauth://totp/...` 链接导入、二维码图片导入，以及 Windows 屏幕框选识别二维码导入。
- 支持双因子条目二维码分享、本地 JSON 备份导出，以及删除/清空前确认提示。
- 新增重复二维码拦截，避免重复添加同一条双因子配置。
- 优化双因子卡片样式，验证码数字加粗，并在最后 5 秒时红色闪烁提醒。
- 重新生成 Windows 安装包与可执行文件。

## 安装包

- `IP诊断工具_0.1.4_x64_zh-CN.msi`
- `IP诊断工具_0.1.4_x64-setup.exe`

## 备注

- 当前双因子密钥仍保存于本机浏览器存储中，适合个人本机使用；如需更高安全级别，建议后续接入系统级加密存储。
